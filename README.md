# Cisco MAC Analyzer

ابزاری برای شناسایی خودکار سوئیچ‌های
Unmanaged در شبکه‌های سیسکو
با استفاده از Python و Netmiko

## مشکلی که حل می‌کند
- شناسایی سوئیچ‌های unmanaged متصل به شبکه
- تفکیک روتر از سوئیچ با CDP
- نمایش پورت‌های مشکوک
- گزارش MAC address و پورت هر دستگاه
- نمایش اینترفیس‌های Down

## ابزارها
- Python 3
- Netmiko
- Cisco IOS

## نمونه خروجی
==================================================
سوئیچ‌های Manageable:
  
  ✅ Gi1/0/2
روترها:
  
  🔵 Gi1/0/1
==================================================
پورت‌های مشکوک به سوئیچ Unmanaged:
 
  ⚠️  Gi1/0/23 (4 MAC)
==================================================
